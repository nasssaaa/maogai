# 刷题宝 · 毛概题库练习（React）

基于 `final_mao_gai_questions.json`（共 508 题，章节 1-8，单选题 + 多选题，**全部自带标准选项和答案**）制作的现代化本地刷题网站。

## 特性

- **练习模式**：顺序 / 随机练习当前筛选范围，支持键盘快捷键（1-6 选答案、方向键翻页、R 随机跳题），做对自动加入「已掌握」
- **考试模式**：可配置题量（默认 30 道单选 + 20 道多选），30 分钟倒计时，支持提前交卷
- **完整答题卡**：考试结束后可查看彩色导航 + 每题你的选择与正确答案高亮对比
- **历史考试记录**：自动保存最近考试，支持一键回放历史答题卡、用相同题目重新考试
- **多用户隔离**：登录后每个用户拥有完全独立的掌握、错题、收藏、答案补充、历史记录（localStorage）
- 自动批改 + 答案补充（支持覆盖题库答案）
- 错题本 & 收藏：一键专项练习
- 题库浏览 + 搜索 + 按章/类型筛选
- 数据统计 + 各章掌握进度可视化
- 答案管理：批量导出 / 导入补充的答案（JSON）
- 黑金主题、完全响应式、零后端依赖、纯前端可直接部署

全部数据保存在浏览器，本地离线可用。

## 快速开始

```bash
cd tiku-brush
npm install
npm run dev
```

打开 http://localhost:5173 即可开始刷题。

## 数据说明

- 原始题目来自桌面 `final_mao_gai_questions.json`（已复制到 `src/data/questions.json`）
- 本题库 **所有 508 题均已包含完整选项和答案**，可直接练习并自动批改。
- “答案补充/覆盖”功能仍然可用，用于用户自定义修正、添加解释，或个人笔记。用户补充的答案优先级高于题库原答案。
- 补充后的答案会保存在浏览器本地，导出后可分享给同学。

## 键盘快捷键（练习中）

- `1` `2` `3` `4` ... 选择对应选项
- `A` `B` `C` `D` 同上
- `←` `→` 上一题 / 下一题
- `空格` 或 `Enter` 下一题
- `R` 随机跳到本会话内另一题
- `S` 提交答案（有标准答案时）

## 构建生产版本

```bash
npm run build
```

产物在 `dist/` 目录，可直接部署到任何静态服务器。

# 刷题宝 · 毛概题库练习（React）

基于 `final_mao_gai_questions.json`（共 508 题，章节 1-8，单选题 + 多选题，**全部自带标准选项和答案**）制作的现代化本地刷题网站。

## 特性

- 练习模式：顺序 / 随机练习当前筛选范围，支持键盘快捷键（1-6 选答案、方向键翻页、R 随机跳题）
- 自动批改：当你为题目补充了选项和答案后，练习时会自动判断对错并高亮
- 答案补充：随时可以为任意题目填写标准选项（支持 2-6 个）和答案（A / AC 等）
- 进度本地持久化：掌握标记、收藏、错题本、补充的答案全部存在浏览器 localStorage
- 错题本 & 收藏：一键将错题/收藏加入专项练习
- 题库浏览 + 搜索 + 按章/类型筛选
- 数据统计：总体 + 每章掌握进度可视化
- 答案管理：批量导出 / 导入补充的答案（JSON），方便备份和多设备同步
- 深色模式、完全响应式、零后端依赖

## 快速开始

```bash
cd tiku-brush
npm install
npm run dev
```

打开 http://localhost:5173 即可开始刷题。

## 数据说明

- 原始题目来自桌面 `final_mao_gai_questions.json`（已复制到 `src/data/questions.json`）
- 本题库 **所有 508 题均已包含完整选项和答案**，可直接练习并自动批改。
- “答案补充/覆盖”功能仍然可用，用于用户自定义修正、添加解释，或个人笔记。用户补充的答案优先级高于题库原答案。
- 补充后的答案会保存在浏览器本地，导出后可分享给同学。

## 键盘快捷键（练习中）

- `1` `2` `3` `4` ... 选择对应选项
- `A` `B` `C` `D` 同上
- `←` `→` 上一题 / 下一题
- `空格` 或 `Enter` 下一题
- `R` 随机跳到本会话内另一题
- `S` 提交答案（有标准答案时）

## 构建生产版本

```bash
npm run build
```

产物在 `dist/` 目录，可直接部署到任何静态服务器。
